## netgear adapter for ioBroker

Remote Control Netgear Switch

## Requirements

-   Node 18 or 20
-   JS-Controller >= 5.0.19
-   Admin >= 6.13.16

## Tested with

-   Netgear JGS524PE (24 Port PoE)
-   Netgear GS108PEv3 (8 Port PoE)
